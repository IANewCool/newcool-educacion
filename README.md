# NewCool Educacion - Educacion Superior Chile

Portal educativo sobre educacion superior y admision universitaria en Chile.

## Demo

[https://educacion.newcool.io](https://educacion.newcool.io)

## Caracteristicas

- Buscador de universidades
- Informacion de becas estudiantiles
- Calculadora de financiamiento
- Proceso de admision PAES
- Beneficios y gratuidad

## Tecnologias

- Next.js 15
- React 19
- Tailwind CSS 4
- TypeScript

## Desarrollo

```bash
npm install
npm run dev
```

## Parte del Ecosistema NewCool

Este modulo es parte de [NewCooltura Informada](https://informada.newcool.io) - Portal de instituciones publicas de Chile.
